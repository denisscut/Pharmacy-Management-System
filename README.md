# Pharmacy Management System

## Introducere

In acest proiect am folosit o arhitectura stratificata (UI / Services(GRASP Controller) / Repository) impreuna cu sablonul Observer.

Aplica»õia permite gestionarea unei liste de medicamente, operatiile posibile sunt: adaugare, stergere, modificare, afisare, cautare, filtrare, sortare si anulare. Este posibila crearea, golirea sau generearea unei retete si exportarea ei (Salvarea retetei intr-un fisier de tip CSV sau HTML).
